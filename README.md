# 2013-06-30 · 华夏资讯网

一个使用**2013年**之前 Web 技术栈复刻的中文门户网站模拟项目。

---

## 项目内容

目前模拟：

- 综合新闻门户首页
- 中国新闻
- 国际新闻
- 科技资讯
- 游戏资讯
- 开发者频道
- 高考专题
- 毕业季专题
- 夏日生活
- 社区论坛
- 早期问答社区

数据全部为 Mock Data，根据[2013年真实事件](https://zh.wikipedia.org/wiki/2013%E5%B9%B4)、互联网文化和当时用户表达方式生成。

### 技术栈

本项目刻意限制在**2013年**左右的Web开发方式。

#### 前端

- HTML
- CSS
- JavaScript ES5
- jQuery 1.10.x

#### 页面结构

- 传统门户网站的信息密度
- 固定宽度布局
- IE8 兼容
- float 布局
- table / ul 结构
- 传统门户三栏结构
- 图片广告位
- 论坛式内容展示
- ES5 JavaScript
- jQuery 1.x
- 手写 CSS

#### 兼容目标

- Internet Explorer 8
- 360 浏览器兼容模式
- **2013年**普通 Windows 用户环境

## 起因

这个项目最开始并不是为了练习前端。  
我小时候经常在家乡的本地 BBS、论坛里浏览各种帖子。那时候的网站没有现在这么复杂，没有无限推荐，没有短视频，也没有那么多算法。  
首页就是一堆蓝色链接、新闻标题、论坛帖子和广告。  
后来学习前端开发以后，我开始好奇：

> “这些十几年前的网站到底是怎么做出来的？”

于是我想，如果现在让 AI 使用当年的技术栈重新制作一个网站，会是什么样？  
所以有了这个项目。

## 为什么选择 **2013年6月30日**

**2013年**是一个很特别的时间点。

那时候：

- 智能手机已经开始普及，但 PC 网站仍然占据主流
- 微信刚刚兴起，微博还处于黄金时期
- iPhone、Android、Windows、浏览器大战仍然激烈
- HTML5、响应式设计、移动互联网正在改变前端
- Bootstrap、jQuery、AngularJS、Node.js 等技术开始影响开发方式

同时，**2013年**上半年发生了很多后来影响深远的事情：

- 社交媒体可以动员的抗议人群
- _斯诺登事件_ 让全球重新讨论互联网隐私
- _棱镜计划_ 改变了人们对网络安全的认识
- _Xbox One 与 PS4_ 开启新一代主机战争
- _Google Reader 关闭_ 象征 RSS 时代逐渐结束
- _iOS 7_ 的扁平化设计改变了移动端视觉语言
- _4G时代即将到来_，移动互联网进入新的阶段
- 互联网公司开始进入新的竞争周期
- 中国领导班子更迭

当时的人并不知道这些事情会影响未来。

他们只是打开浏览器，看新闻，刷论坛，下载软件，等待游戏更新。

## 那个夏天

**2013年**，我还是一个小学生，那时候的夏天我仍有印象：

没有现在这么多智能设备生态，没有人人都在讨论 AI，也没有短视频占据所有碎片时间。  
_《环太平洋》_、_《战地4》_ 、iPad 上的游戏、麻将馆的空调、小店门口的弹珠机……  
大人们还在使用 QQ、微博、论坛，讨论房价、工作、孩子的成绩。  
那是一个现在回看很普通，但又正在发生巨大变化的时代。  
经济仍然保持高速增长，很多家庭对未来充满期待。  
很多今天已经成为过去的东西，在那个夏天都还鲜活。

> 夏天对于中国的孩子是特殊的，  
> 中考，高考，毕业，考研，进入社会之前所有重大人生转折都发生在夏天。  
> 无处可藏的炎热意味着过往的终结和新生活的开启，逼迫着对成熟毫无心理准备的我们往前走。

## 一些启发

在做这个项目的之前，我看到了一篇关于 2013年大学生活和互联网创业回忆 的[文章](https://x.com/chunxiangai/status/1861048967800197180)，作者写到：

> 那是一个蓬勃时代的前夜，无知的少年在食堂里抢着他的煎蛋挂面，做什么事都猴急，生怕有什么没有体验到，没有玩过。却错过了身边一场又一场奇迹。

这段话让我想到：

技术变化不仅仅是版本升级，每一代互联网背后，其实都有一群人在那个时间点生活。  
有人研究单片机、有人写第一个 App、有人在论坛发帖、有人在网吧通宵；他们当时不知道自己经历的是一个时代的转折。

## 最后

这个项目没有什么实际商业价值，它更像一次互联网考古。
现在打开一个十年前的网站，可能会觉得粗糙、拥挤、不够漂亮，但是那些页面曾经承载过很多人的青春：
